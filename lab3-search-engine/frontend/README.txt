DESCRIPTION
  An image search engine web frontend which supports width and height range
  filter, color filter and image number limits.

TECHNOLOGIES
  react, react-dom, react-route-dom, react-redux, chakra-ui

PREREQUISITES
  - A running backend server (see ../backend)
  - Install dependencies in package.json (npm install)

RUN
  - Edit .env
  - Run npm start

DOCKER
  See Dockerfile.
